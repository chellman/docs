Design basics

Macaw allows for workflow that is similar to that of an image editor. You can draw and move elements freely on the canvas, then style them as you see fit. After that, if good HTML is important to you, you can organize and describe elements in the outline panel. The more Macaw knows about your elements purpose the better code output it will produce.

What's set's Macaw apart from other design software is it's ability to define elements dimensions both in pixels or percentages.

The Grid

Grid is an essential part of Macaw's workspace and workflow. It helps you to place elements evenly on the page and serves as a guide throughout the whole design process.

You can set up the grid to match your preferences, using Grid Width, Number of Grid Columns and Space Between Grid Columns. Like many other elements grid width can be set either as in percentages or pixel dimensions.

Drawing

Primary way of creating content in Macaw is by drawing. To draw some elements just pick the desired tool from the tool-bar then click + drag on the canvas. There are no constraints on where to draw an element, you can draw anywhere on the canvas, including on top of other elements. You can draw text elements, containers, buttons or even form elements. Images can be placed on the canvas right from within the library.

You can draw elements with percentage or pixel based widths and positions. These can be also adjusted later.

While drawing, elements will snap to grid, smart guides, canvas boundaries or other elements.

Elements

Elements can be drawn with the Element Tool R, when converted to code they will become block level elements such as divs, sections, forms, etc.

Containers

Design wise containers are used for grouping other elements, but unlike groups in standard design software, containers have their own dimensions (or even paddings) and can be styled just like standard elements.

You can create containers by using the Elements > Group command. Another way of creating containers is by using the Container Tool G, with it selected simply drag over some elements to wrap them in a container. Groups created with Container Tool will immediately have dimensions based on the area you've dragged over while drawing, whereas grouping with Elements > Group command will adjust the group size to fit it's children.

If among the elements that you are grouping together with Elements > Group command, one seems to fit the task of containing all the other elements, Macaw will ask you if you want to transfer presentation attributes of that element to the newly created group. In other words: Macaw is smart enough to use existing elements as groups when it makes sense, and leave you with cleaner code in the process.

Text

Macaw has the advantage of using a browser based canvas, which allows the text to be rendered exactly like it will be in the target browser.

Paragraph and Point text fields

You can create two types of text boxes in Macaw. Simply clicking on the canvas with the Text Tool selected, will create a Point Text field that has auto width and behaves like an inline element.

To create a block based text box, that has set dimensions and behaves like a block level element, click and drag on the canvas with Text Tool selected.

You can also toggle between two types of text boxes after you've created a field, by choosing an adequate value from width field's units in Dimensions section of Inspector Pane. Choosing Auto will set the text box to Point Text, whereas px or % will set the text box to Paragraph Text.

Paragraphs versus line breaks

Hitting Return/Enter to separate lines of text inside a text box will result in code output for that box being divided into several paragraphs. You can also use Shift + Return/Enter to create line breaks (<br>) in the code output.

Fast text prototyping

Macaw supports Lorem Ipsum text generation, to quickly fill text boxes with sample copy. To add some generated text to a text box, just double click the text box, type loremXp (where X stands for number of paragraphs you wish to create) then hit TAB and Macaw will generate X paragraphs of Lorem Ipsum….

For finer amounts of generated text, you can type loremXw instead (where X stands for number of words you wish to add) then hit Tab and Macaw will add X words to that text box.

Both shortcuts can be used multiple times in one text box, to fill it with sample content just as you see fit.
